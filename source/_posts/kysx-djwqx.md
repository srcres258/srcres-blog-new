# 常用等价无穷小收集

记录我学习考研数学知识, 以及刷考研习题集以来遇到过的所有等价无穷小类型.
(目前以来遇到过的. 如果有新的会及时补充.)

重要等价无穷小已用★标出. ★越多表示越重要.

## 基本 (来自网上总结)

$\sin x \sim x$

$\arcsin x \sim x$

$\tan x \sim x$

$\arctan x \sim x$

$\ln (1 + x) \sim x$

$\mathrm e^x - 1 \sim x$

$x - \sin x \sim \frac{1}{6} x^3$

$\arcsin x - x \sim \frac{1}{6} x^3$

$\tan x - x \sim \frac{1}{3} x^3$

$x - \arctan x \sim \frac{1}{3} x^3$

$\arcsin x - \arctan x \sim \frac{1}{2} x^3$

$\tan x - \sin x \sim \frac{1}{2} x^3$

$1 - \cos x \sim \frac{1}{2} x^2$

$x - \ln (1 + x) \sim \frac{1}{2} x^2$

★ $a^x - 1 \sim x \ln a$

$1 - \sqrt{1 - x^2} \sim \frac{1}{2} x^2$

## 网上没有但是来自教辅

★ $(1 + x)^a \sim 1 - ax$

## 学习时自行推广, 得到

$(1 + x)^\frac{1}{x} - \mathrm e \sim \frac{\mathrm e}{2} x$ ($x \to 0^+$)
